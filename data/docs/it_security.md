# IT Security Guidelines

**1. Password Rules**  
Passwords must contain at least 12 characters and be changed every 180 days. Multi-factor authentication is mandatory.

**2. Device Usage**  
Company devices may only be used for business purposes. Lost or stolen devices must be reported immediately.

**3. Data Protection**  
Confidential data must not be stored on personal devices or shared via unauthorized channels.

**4. Incident Reporting**  
Security incidents must be reported to the IT security team within 24 hours.
